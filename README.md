##基础架构
>react + mobx + webpack + scss 


## 开发说明

- master分支作为稳定版本  dev为上线前打包构建分支
>开发人员基于dev拉取分支，分支命名规范dev-名字缩写  eg:dev-fff
> 运行 npm start 启动本地服务  参数（port：指定端口号，target：指定代理）
> 运行 npm run mock 统一走mock代理接口
